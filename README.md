About `census.stat`
-------------------

`census.stat` stores variables downloaded via `tidycensus`.

Installation
------------

``` r
# install.packages("devtools")
devtools::install_github("jjchern/census.stat@v0.0.2")

# To uninstall the package, use:
# remove.packages("census.stat")
```
